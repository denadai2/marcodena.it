Cartolina

Cartolina is a personal project which allows you to send postcards directly from your iPhone. It allows you to take a picture of a place and send a physical postcard to your family and friends. This allows you to share real emotions, not pre-built pictures. I developed the application, the design and the API system. The first one was made for iOS 6/7, using: AFNetworking, CrittercismSDK, Foursquare-API and TestFlight. The API system is built with Python and Flask.
